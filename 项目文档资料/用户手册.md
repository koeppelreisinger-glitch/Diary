# Echo AI 日记应用用户手册

## 1. 产品简介

Echo AI 是一款智能日记应用。你不需要每天写一篇完整日记，只要像聊天一样告诉 Echo 今天发生了什么，系统就会帮你整理成日记，并自动提取事件、情绪、消费、地点和灵感，方便以后回忆和搜索。

## 2. 访问入口

本地运行时，常用入口如下：

| 功能 | 地址 |
|---|---|
| 正式用户入口 | `http://localhost:8000/frontend/login.html` |
| 今日记录页 | `http://localhost:8000/frontend/today.html` |
| 回忆页 | `http://localhost:8000/frontend/index.html` |
| 寻迹页 | `http://localhost:8000/frontend/trace.html` |
| 开发调试入口 | `http://localhost:8000/frontend/debug.html` |
| 后端联调台 | `http://localhost:8000/frontend/test-console.html` |

普通用户建议从正式用户入口进入。

## 3. 注册与登录

### 3.1 注册账号

1. 打开登录页。
2. 点击“注册”。
3. 输入手机号、昵称和密码。
4. 点击注册按钮。
5. 注册成功后，系统会自动登录并进入应用。

### 3.2 登录账号

1. 打开登录页。
2. 输入手机号和密码。
3. 点击登录。
4. 登录成功后进入今日记录页面。

如果页面自动跳回登录页，说明登录状态已失效，需要重新登录。

## 4. 今日记录

今日记录是应用的核心功能。

### 4.1 开始记录

1. 进入今日页。
2. 点击“开始今日记录”。
3. 在输入框中写下今天发生的事情。
4. Echo 会根据你的内容进行回应，并引导你继续补充。

你可以记录：

- 今天做了什么事。
- 今天心情怎么样。
- 花了多少钱，买了什么。
- 去了哪里。
- 有什么想法或灵感。

示例：

```text
今天上午开了两个会，有点累。中午买了一杯咖啡花了 32 元，下午和朋友聊了很久，心情变好了。
```

### 4.2 继续补充

如果当天还有新的事情，可以继续输入。系统会把这些内容都保存到今天的会话中。

### 4.3 结束记录

1. 点击“结束今日记录”。
2. 系统开始生成日记。
3. 等待几秒后刷新或查看今日页。
4. 生成完成后，可以看到完整日记、摘要和结构化信息。

注意：AI 生成可能需要几秒到几十秒。如果看到“生成中”，等待后再刷新即可。

## 5. 查看日记

生成完成后，今日页会展示：

- 日记正文
- 摘要
- 情绪分数
- 关键词
- 事件
- 情绪
- 消费
- 地点
- 灵感

这些内容由 AI 根据聊天记录自动整理，用户可用于当天复盘。

## 6. 回忆功能

进入回忆页后，可以查看过去的日记记录。

你可以：

- 按时间查看历史日记。
- 查看某天的完整详情。
- 通过关键词搜索相关记录。
- 查看日历中的记录分布。

搜索示例：

```text
咖啡
会议
生命短暂
```

搜索结果会展示与关键词相关的日记内容，点击后可进入详情页。

## 7. 寻迹功能

寻迹页会把日记中的结构化信息单独展示，帮助你从不同角度回看生活。

主要包括：

| 分类 | 说明 |
|---|---|
| 事件 | 最近发生过的重要事情 |
| 情绪 | 记录中的情绪标签和强度 |
| 消费 | 日记中提到的消费金额和分类 |
| 地点 | 记录中出现过的地点 |
| 灵感 | 生活中的想法、感悟和灵感 |

例如，你可以通过寻迹页看到最近花了哪些钱、经常出现什么情绪、去过哪些地方。

## 8. 图片功能

在支持图片上传的页面中，可以上传当天相关图片。图片会和日记记录关联，用于日后回忆。

注意：

- 本地运行时图片会保存到项目的上传目录。
- 线上部署时，如果没有配置对象存储，图片可能无法长期保存。

## 9. 个人中心

个人中心用于查看或修改个人信息和偏好设置。

常见设置包括：

- 昵称
- 时区
- 输入偏好
- 提醒设置
- 深色模式等

建议根据所在地区设置正确时区，否则“今天”的日期归属可能不准确。

## 10. 开发联调入口

如果是开发者或测试人员，可以使用：

```text
http://localhost:8000/frontend/debug.html
```

或：

```text
http://localhost:8000/frontend/test-console.html
```

联调台适合：

- 快速注册测试账号。
- 登录并查看 Token。
- 测试后端接口。
- 查看原始接口响应。
- 排查注册、登录、生成日记等问题。

普通用户不需要使用联调台。

## 11. 常见问题

### 11.1 为什么结束记录后没有马上出现日记？

日记生成需要 AI 处理，可能需要等待一段时间。看到“生成中”时，稍等几秒后刷新即可。

### 11.2 为什么打开页面后回到登录页？

说明当前没有登录状态，或 Token 已失效。重新登录即可。

### 11.3 为什么搜索不到内容？

可能原因：

- 还没有生成过相关日记。
- 输入的关键词和记录内容不一致。
- 日记仍在生成中。

### 11.4 为什么图片线上显示异常？

线上环境如果没有对象存储，图片不会稳定保存。需要接入云存储后才能完整支持线上图片回忆。

### 11.5 为什么日期不对？

请检查个人设置中的时区。时区会影响今日记录归属到哪一天。

## 12. 推荐使用流程

1. 每天打开今日页。
2. 用几句话记录当天发生的事情。
3. 有消费、地点、情绪或灵感时直接写出来。
4. 晚上点击结束记录。
5. 查看 AI 生成的日记。
6. 过几天通过回忆和寻迹功能复盘生活。
