# 作品信息概要表

| 项目 | 内容 |
|---|---|
| 作品名称 | Echo AI 日记应用 |
| 作品类型 | AI 智能日记 / 个人记录 / 情绪回顾工具 |
| 作品简介 | Echo AI 是一款通过聊天方式记录生活的智能日记应用。用户只需像聊天一样输入当天经历，系统会自动生成完整日记，并提取事件、情绪、消费、地点和灵感等结构化信息，方便后续回忆、搜索和复盘。 |
| 目标用户 | 想记录生活但难以坚持写长日记的人；关注情绪管理、自我成长和生活复盘的人；希望用 AI 整理个人记忆的人。 |
| 核心功能 | 用户注册登录、今日对话记录、AI 日记生成、历史回忆、关键词搜索、寻迹分析、图片上传、个人设置、开发联调入口。 |
| 技术特点 | 采用 FastAPI 构建后端接口，SQLAlchemy 管理数据库，Pydantic 进行数据校验，前端使用 HTML/CSS/JavaScript 实现，接入 TokenHub AI 能力完成智能对话和日记生成。 |
| 主要优势 | 使用门槛低，聊天即可记录；AI 自动整理内容，节省写作时间；支持结构化提取，便于搜索和分析；前后端结构清晰，方便维护和扩展。 |
| 后端技术 | Python、FastAPI、Uvicorn、SQLAlchemy、asyncpg、Pydantic、python-jose、bcrypt、httpx、Pillow |
| 前端技术 | HTML、CSS、JavaScript、Fetch API、本地 Token 存储 |
| 数据库 | PostgreSQL |
| AI 服务 | TokenHub Chat Completions / GLM 模型 |
| 部署方式 | 本地运行或 Vercel 部署 |
| 本地访问入口 | `http://localhost:8000/frontend/login.html` |
| API 文档入口 | `http://localhost:8000/docs` |
| 健康检查入口 | `http://localhost:8000/health` |
| 项目状态 | 已完成 MVP 核心闭环，可进行演示、联调和继续扩展 |
